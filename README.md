Репозиторий Кузнецова Ильи с кодом реализации нереволюционного алгоритма SANE для обучения полносвязной нейросети с одним скрытым слоем.

Для запуска кода необходимо создать виртуальное окружение и установить зависимости с помощью poetry (команды poetry init и install).

В файле main.py представлена функция для запуска основного цикла алгоритма SANE. Для корректной работы программы нужно в проекте создать папки для сохранения моделей (model) и графиков (plot).

Для получения графика изменения усредненных значений потерь и точности с нескольких эксаериментов необходимо запустить файл plot_experiments.py.
